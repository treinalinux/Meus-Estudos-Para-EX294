# Meus Estudos para o EX294

Objeto do repositório é registrar meus estudos para a certificação RHCE EX294 V9. Eu quero colocar aqui os comandos que achar interessante para me lembrar durante o exame.

Para que ainda não conhece o que pedido, pode olhar na página oficial da Red Hat que lá sempre vai ter o conteúdo atualizado. O que posso dizer é que são 4 horas de prova, isso parece muito se você tiver já acostumado com o ansible ou bem preparado para o exame, se não estiver, é melhor reagendar. É de doer os ossos, nos exames de 3 horas, então 4 horas, é muito cansativo.

Os requesitos de dispositivos se você não conseguir agendar para um centro é um computador ou laptop de no mímino 8G, mas recomendo ter o laptop apenas como um backup caso falta luz, tente fazer no computador com no mínimo 16G, com um monitor de no mínimo 24 polegadas. O melhor seria mesmo fazer o exame em um centro de testes, mas se não tiver jeito, use o computaodr ponte e um monitor grande, porem estude tanto pelo computador e pelo laptop.

Você precisa simular o que é pedido no exame, tipo, se tiver faltando 3 semanas para o exame, deve simular o exame 3x (1x por semana), simular mesmo as 4 horas com os intervalos de 5 minutos para ir ao banheiro, ou esticar mesmo as pernas, vai perceber que no dia seguinte a simulação, estará exausto. Por isso, é necessário simular. Se estiver sedendário (o meu caso), precisa tentar ao menos fazer algumas caminhadas (melhor procurar orientações de um médico para melhor sua resitência para o exame).

A simulação vai te ajudar a medir o tempo, tenta dividir 4 horas por um número de quesões de acordo com o que é pedido.

Você precisa simular, você treinar, as provas da Red Hat, são práticas, então deve treinar bastante, até onde sei desde algum momento do ano passado foram disponibilizados retakes (veja se isso ainda está válido no site). Tendo retake, pode treinar e simular como eu tinha falado acima, e se reprovar vai ter uma segunda chance, mas estude para passar na primeira.

Se tiver condições financeiras, então, faça o exame oficial.

Acho que agora no momento o ainda estou com dificuldades de entender é usar o **navigator**. Sinceramente ainda não entendi o motivo de ter que usar isso **navigator**.
